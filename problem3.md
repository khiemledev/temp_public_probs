# Kiểm tra DSLK đơn đối xứng

Cho danh sách liên kết đơn, kiểm tra xem danh sách có đối xứng hay không.

**Input:**

- Dòng đầu tiên là `n` số nút
- Dòng tiếp theo là `n` node trong DSLK đơn

**Output:**

- In ra "CO" hoặc "KHONG"

**Ví dụ:**

Ví dụ 1:

```
# Input
5
1 2 3 4 5

# Output
KHONG
```

Ví dụ 2:

```
# Input
6
0 1 2 2 1 0


# Output
CO
```
